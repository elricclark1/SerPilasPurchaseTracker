# Project Changelog

## 2026-01-02
- **Unified Codebase:** Merged `app.py` and `local_app.py` into a single `app.py`.
- **Environment Logic:** Implemented `IS_LOCAL_MODE` to toggle between Web (ephemeral) and Local (persistent) modes.
- **Persistence:** Local mode now saves data to CSV and settings (accent color, paths) to `settings.json`.
- **UI Overhaul:**
    - Adopted "HabitKit" aesthetic (Pitch Black background, high contrast cards).
    - Moved Help button to top-right.
    - Switched View selector to a Toggle switch.
    - Increased Transaction Log height to 800px.
- **Charts:**
    - Pie Chart: Added percentage labels (`textinfo='percent+label'`) and increased font size.
    - Bar Chart: Added values inside bars (`textposition='inside'`).
- **Features:**
    - Added "Custom Days" option to time filters.
    - Implemented bulk deletion in Transaction Log.
    - Removed legacy `N` column from schema.
## 2026-01-02 (Update 2)
- **UI/UX Expansion:**
    - Moved "Download Source" into sidebar expander to save vertical space.
    - Replaced Category multiselect with a **500px scrollable checkbox list** for rapid filtering.
    - Added **"Select All" and "Clear"** buttons for categories (callback-based fix).
    - Expanded metrics to **4 columns**; added **"Avg. Spent per Month"**.
    - Increased metric font size to **36px** for high visibility.
    - Added preset time filters: **14, 60, 90, 180, and 365 days**.
- **Documentation & Infrastructure:**
    - **Git Integration:** Initialized repository, configured SSH authentication, and pushed to GitHub.
    - **README Fix:** Removed hardcoded local paths and standardized installation steps.
    - **Help Dialog:** Expanded explanation of Web vs. Local modes and privacy philosophy.
- **Notes:** Created `Git_Setup_Notes.md` for local reference.
